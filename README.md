# Shopping Cart - Phone Store 🛒

A C++ console application simulating a phone shopping cart system.

## Features
- Add and remove phones from cart
- Undo last delete using a Stack
- Apply promo codes for discounts
- Display cart with total price

## Data Structures Used
- Linked List (shopping cart)
- Stack (undo/restore system)

## How to Run
g++ main.cpp -o cart
./cart
